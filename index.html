What is Hybrid Framework for HTML5 ?

Want single code through out multiple platforms - go HTML5 but don't go full way, go half way.
Want full controll and all you can imagine accomplished in your app - go native, buto don't go full way, go half way.
Brind both halves together and:
  • you are cost saving all the way
  • you are developing more easy and with more opportunitites all the way
  • you are bringin your app to multiple platforms faster all the way
  • you are happier and more successfull all the way
The Hybrid Framework for HTML5 applications for iOS and Android enables you to wright HTML5 applications which can be installed on a device as a native application. The HTML5 application runs in a container provided by the framework.
For short we abreviate the Hybrid Framework for HTML5 mobile applications as HFH.
